## Repository Organization

Some files may appear duplicated because certain problems are referenced across multiple platforms.

To simplify navigation, the file:

`z - Complete List`

contains an indexed list of solved problems **without duplicates**, with the platform name included at the start of each file name.

Example:

```text
AtCoder - Problem Name.cpp
Codeforces - Problem Name.cpp
Olympiads - Problem Name.cpp
```

For USACO problems in the complete list, the prefix indicates the section of the USACO Guide:  
`1B` = Bronze  
`2B` = Silver  
`3B` = Gold  
`4P` = Platinum
